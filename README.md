## 🗓 진행 방식
- **진행 기간:** 202X년 2월 11일 ~ 
- **사용 언어:** Java, Python
- **업로드 마감:** 매주 일요일 전까지 Pull Request (PR) 생성
- **확인 및 병합:** 매주 일요일 (담당자: 박성은)

## 🗂 참여 방법 (Sync Fork & PR)
1. 이 레포지토리를 자신의 계정으로 `Fork` 합니다.
2. 매주 작업을 시작하기 전, 자신의 GitHub 레포지토리에서 **`Sync fork`**를 눌러 최신 상태로 동기화합니다.
3. 로컬로 `pull`을 받은 후, 주차별 폴더(`Week_XX`)에 자신의 이름으로 된 폴더를 생성합니다.
4. 본인의 언어(Java 또는 Python)로 공부한 코드와 마크다운 메모 등을 작성하여 커밋합니다.
   - 📂 경로 예시: `Week_01/홍길동/study_content.py`
5. 자신의 레포지토리에 Push 한 후, 이 원본 레포지토리로 **Pull Request**를 보냅니다.
6. 일요일에 담당자가 확인 후 Merge 합니다.
